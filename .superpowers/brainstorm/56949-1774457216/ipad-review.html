Key Insights for iPad Pro 13"

  • Split Analysis: The 13" screen can easily show both sensor sets side-by-side, which is the core value proposition of a "Lab" app. No more scrolling to compare A vs B.
  • Persistent Navigation: We can use a NavigationSplitView or a sidebar on the iPad instead of a bottom TabBar, which is more idiomatic for iPadOS.
  • Reachability: Controls should be clustered in a way that's easy to reach if the iPad is mounted on a trainer stand.
